About StockEye / Product Description

StockEye provides a streamlined web-based platform for beginner and experienced investors to view popular stocks based on Twitter sentiment analysis. The user may also create an account, allowing said user to follow the Twitter sentiment of desired stocks on their own feed. This new platform will save the user valuable time as they will no longer be required to research and analyse the Twitter sentiments manually.

Twitter sentiments analysis web application

Stockeye is a web application that looks into well known stocks, as it classifies the stock’s data using twitter sentiments, that is shared to users. Users interacting with these products, include stockholders, and investors, that are looking for an efficient way to buy, and trade stocks, within a controlled environment. The application presents a watch list of stocks to users, which they are then able to track its sentiments. This is useful for clients,by saving time, as investors aren’t required to go and identify data on their own.

Description of Users

The user base StockEye intends to serve are people who want a time-efficient way of gathering information about stocks they plan to invest in. Users can range from beginners to financial experts as there is no complex prerequisite understanding of the stock market required. However, they must be familiar with basic website navigation in order to use the application.
